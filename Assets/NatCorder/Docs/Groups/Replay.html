Replay



Description

This is a lightweight API for quickly recording the screen.



Static Functions

PauseRecording Pause recording.
ResumeRecording Resume recording.
StartRecording Start recording a replay with optional audio.
StartRecording Start recording a replay with audio from an AudioListener.
StartRecording Start recording a replay with audio from an AudioSource.
StopRecording Stop recording a video.


Static Variables

IsPaused Is recording paused?
IsRecording Is a video being recorded?
