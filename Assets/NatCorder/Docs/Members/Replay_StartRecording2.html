Replay.StartRecording

public static void StartRecording (Camera recordingCamera, Configuration configuration, VideoCallback replayCallback, AudioSource audioSource, bool audioIsRecordOnly);

Description

Start recording a replay with audio from an AudioSource.

When the audioIsRecordOnly flag is enabled, the audio source will be muted once the audio samples have been committed for encoding. This is useful for recording audio sources without hearing them in game, like when recording microphone audio.

See Also: Configuration , VideoCallback .
