Sharing.GetThumbnail

public static void GetThumbnail (string path, Action`1 callback, float time);

Description

Get a thumbnail texture for a recorded video.

You can specify a time at which to retrieve the thumbnail. Note that requesting a time that is not within the range of the duration of the video is undefined behaviour. 

Sharing.GetThumbnail(path, delegate (Texture2D thumbnail) {
    rawImage.texture = thumbnail;
});

See Also: VideoCallback .
